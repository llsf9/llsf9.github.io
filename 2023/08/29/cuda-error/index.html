CUDAに関するエラーの整理

RuntimeError: CUDA error: XXX [BLABLABLA].CUDA kernel errors might be asynchronously reported at some other API call,so the stacktrace below might be incorrect.For debugging consider passing CUDA_LAUNCH_BLOCKING=1.

ゼミ研究中に初めてこのエラーに遭った、
エラーが繰り返すとGPUがダウン可能性もある！

原因

  • 直接原因：CPUとGPUの処理
    https://blog.csdn.net/weixin_43590796/article/details/131588761
  • 間接原因：index関連のコードが間違っている[out of boundとか]

    先輩回答：
    このエラーはGPUメモリ上に載せたテンソルについてIndexを間違えた時によく見かけます
    データセットの画像サイズが変わったとか、Indexが関わりそうなデータセット間の違いが原因かなという気がします
    このエラーなんですが、繰り返すとGPUが一時的に使えなくなるので、一旦CPU上で動かしてデバッグした方が良いかも(エラーメッセージのタイミングも実際のエラーが起きている箇所と違うところになったりするので、↑のスクショのRunTimeErrorもあまり当てにならない)

影響
サーバーGPUのダウン【気をつけなきゃ！】

解決
https://discuss.pytorch.org/t/cuda-runtime-assertion-error-index-out-of-bound/140173

  1. ‘CUDA_LAUNCH_BLOCKING=1’
  2. CPUモードで様子を確認（’CUDA_VISIBLE_DEVICES=-1’）
